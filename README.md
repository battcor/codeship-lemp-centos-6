# Quickstart

```
git clone https://github.com/battcor/codeship-lemp-centos-6
cd codeship-lemp-centos-6/
jet steps
docker run -p 127.0.0.1:8080:80 battcor/docker-lemp-centos-6
```

Then, open http://127.0.0.1:8080 in your browser
